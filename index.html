Safety culture survey

Setup required before the survey can be used

Connect to Supabase

1

Create a Supabase project

Go to supabase.com, sign up free, and create a new project. Note down your project URL and anon key from Project Settings → API.

2

Create the database table

In Supabase go to SQL Editor and run this query:

create table survey_responses (
  id bigint generated always as identity primary key,
  created_at timestamptz default now(),
  team text not null,
  avg_score numeric not null,
  dim_avgs jsonb,
  answers jsonb,
  comment text
);

alter table survey_responses enable row level security;

create policy "Allow insert" on survey_responses
  for insert with check (true);

create policy "Allow select" on survey_responses
  for select using (true);

create policy "Allow delete" on survey_responses
  for delete using (true);
3

Add your credentials to this file

Open this HTML file in any text editor (Notepad, TextEdit, VS Code). Find the section near the top of the <script> that says SUPABASE CONFIGURATION and replace the two placeholder values with your URL and anon key. Save the file.

4

Deploy to Vercel

Rename the file to index.html, put it in a folder, and drag that folder to vercel.com/new. Your survey will be live at a yourname.vercel.app URL.
